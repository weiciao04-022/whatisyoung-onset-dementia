01 / 什麼是年輕型失智症

疾病來得比
想像中更早

有些病患在 65 歲之前便已發病，我們稱之為早發性失智症（Early-onset dementia），或年輕型失智症（Young onset dementia）。
TYPE 01↯

血管性失智症

腦血管疾病造成認知功能受損。

TYPE 02◒

額顳葉型失智症

早期可能出現行為、人格或語言改變。

TYPE 03＋

可治療型失智症

由可逆因素引起，需及早辨識與治療。

TYPE 04∞

阿茲海默症

記憶與認知功能逐步退化。

03 / 配套問題

年輕，反而
找不到位置

既有長照體系多以高齡者為想像，年輕型失智症患者的需求，經常掉進制度縫隙。

照顧據點太少

服務距離與交通成本，讓持續參與成為負擔。

照顧需求不同

他們可能仍有工作、育兒與社會參與需求，並不等同高齡照顧。

收案標準困難

年齡、失能程度或診斷條件，使有需求的人未必進得了服務系統。

04 / 工作權

確診之後，
還能工作嗎？

「能力改變了，
不代表價值消失了。」

對仍在工作年齡的患者而言，工作不只是收入，也是身分、尊嚴與社會連結。需要被調整的是職務與環境，而不是立刻將一個人排除。

01

合理調整

拆解任務、調整工時，保留熟悉且能完成的工作。

02

友善理解

讓雇主與同事理解症狀，降低誤解、標籤與排擠。

03

轉銜支持

在疾病進程中，協助患者從原職務平順轉向適合的參與方式。
